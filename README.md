# ESP32 BLE guitar

Transform toy guitar into a Clone Hero controller with ESP32

**Pinout**

| ESP32 GPIO | Button               |
| ---------- | -------------------- |
| GPIO36     | Whammy potentiometer |
| GPIO32     | Strum up             |
| GPIO33     | Strum down           |
| GPIO21     | SDA                  |
| GPIO22     | SCL                  |
| 3.3V       | V pin on neck        |
| GPIO25     | LED on DPAD          |
| GPIO23     | Start                |
| GPIO24     | Select               |
| GPIO19     | DPAD RIGHT           |
| GPIO18     | DPAD LEFT            |
| GPIO17     | DPAD DOWN            |
| GPIO16     | DPAD UP              |
